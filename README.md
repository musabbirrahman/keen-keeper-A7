# KeenKeeper 🤝

**KeenKeeper** is a modern relationship management dashboard designed to help you stay consistent with your social connections. By setting contact goals and logging interactions, users can ensure they never lose touch with friends, family, or mentors.

## 📝 Project Description
In a busy world, it’s easy to let months pass between catch-ups. KeenKeeper acts as a "personal CRM" for your social life. It provides visual feedback and data-driven analytics to empower you to be more intentional and consistent with your most important relationships.

## ✨ Top Features
* **Friendship Dashboard**: A high-level overview of all connections featuring smart status badges (**On-Track**, **Almost Due**, or **Overdue**) based on custom contact goals.
* **Quick Check-In System**: Log interactions—including **Calls**, **Texts**, **Video Chats**, and **Meetups**—with a single click directly from a friend's profile.
* **Global Activity Timeline**: A centralized, filterable history that tracks every interaction you've made across your entire network, powered by a global React Context state.
* **Friendship Analytics**: Beautiful, interactive donut charts powered by **Recharts** that visualize your interaction habits by type in real-time.

## 🛠️ Technologies Used
* **Framework**: React.js (Vite)
* **Routing**: React Router v6 (using Loaders and dynamic routing)
* **State Management**: React Context API
* **Styling**: Tailwind CSS & DaisyUI
* **Data Visualization**: Recharts
* **Icons**: React Icons (Feather Icons)
* **Notifications**: React Toastify
* **Deployment**: Vercel
