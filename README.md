## 📖 About The Project
In a busy world, it’s easy to let months pass between catch-ups with the people who matter most. **KeenKeeper** acts as a lightweight relationship manager. By setting contact goals and logging your interactions, the dashboard provides visual feedback and analytics to empower you to be more intentional with your friendships, family, and professional network.

### ✨ Key Features
- **📊 Interactive Dashboard:** A high-level overview of all connections featuring smart status badges (*On-Track, Almost Due, Overdue*) calculated dynamically from your contact goals.
- **⚡ Quick Check-Ins:** Log interactions—including *Calls*, *Texts*, *Video Chats*, and *Meetups*—with a single click directly from a friend's profile.
- **⏱️ Global Activity Timeline:** A centralized, filterable history that tracks every interaction across your network, synchronized via React Context.
- **📈 Friendship Analytics:** Beautiful, interactive donut charts powered by Recharts that visualize your interaction habits in real-time.
- **🔔 Smart Notifications:** Instant, polished UI feedback utilizing React Toastify.

**Frontend & Core**
- **[React 18](https://react.dev/)** - UI Library
- **[Vite](https://vitejs.dev/)** - Build Tool & Bundler
- **[React Router v6](https://reactrouter.com/)** - Client-side routing & data loading (`useLoaderData`)
- **React Context API** - Global state management

**Styling & UI**
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first styling
- **[DaisyUI](https://daisyui.com/)** - Tailwind component library
- **[Recharts](https://recharts.org/)** - Composable charting library
- **React Icons** - Lightweight SVG icon library (Feather Icons)
- **React Toastify** - Alert & notification system
